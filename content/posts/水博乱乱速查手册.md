# 水博乱乱速查手册

> 快速查找关键概念、信号和操作规则。

---

## 📘 核心概念速查

| 概念 | 定义 | 用途 |
|------|------|------|
| **订单簿** | 所有未成交限价单的集合 | 看挂单压力/支撑 |
| **买盘 (Bid)** | 买入挂单，价格下方 | 支撑力量 |
| **卖盘 (Ask)** | 卖出挂单，价格上方 | 压力来源 |
| **挂单墙 (Wall)** | 某价位异常大量挂单 | 关键支撑/阻力 |
| **Spoofing** | 虚假挂单，到价就撤 | 需排除假墙 |
| **CVD** | 累积(主动买量-主动卖量) | 真实买卖力量 |
| **Delta** | 单时段(主动买-主动卖) | 短期力量 |
| **POC** | 成交最密集的价格 | 成本平衡点 |
| **VAH** | 成交集中区上沿 | 做空参考 |
| **VAL** | 成交集中区下沿 | 做多参考 |
| **AVWAP** | 从某时点算的量价均线 | 平均持仓成本 |
| **OI** | 未平仓合约总量 | 市场参与度 |
| **SFP** | 假突破（Swing Failure） | 反转信号 |
| **GEX** | 期权 Gamma 风险敞口 | 波动方向 |
| **CME 缺口** | CME 收盘/开盘价差 | 回补交易 |

## 🔵 蓝色信号系统

| 项目 | 说明 |
|------|------|
| **触发条件** | 订单簿买卖差值 ≥ 5% |
| **位置** | 通常在前低下方 |
| **含义** | 大量现货+合约买单涌入 |
| **胜率** | 中线 85%+ |
| **时间框架** | 中线（数天） |
| **注意** | 非 100%，需配合入场模型 |

## 📊 CVD 速查

| CVD 状态 | 含义 | 操作 |
|----------|------|------|
| CVD 上升 + 价格上升 | 真涨，多头强 | 持多 |
| CVD 下降 + 价格下降 | 真跌，空头强 | 持空 |
| CVD 下降 + 价格上升 | **虚涨**，卖压实际更强 | 警惕 / 做空 |
| CVD 上升 + 价格下降 | **虚跌**，买盘暗中吸筹 | 准备做多 |
| CVD 归位/重置 | 失衡已消化 | 平仓时机 |

**CVD 止盈法**：按 CVD 归位进度分批止盈（如归位 80% → 平仓 80%）

## 🎯 入场模型 Checklist

进场前确认：

- [ ] **订单簿**：关键位有挂单支撑/压力？
- [ ] **CVD**：方向一致？有无背离？
- [ ] **Volume Profile**：在 POC/VAH/VAL 关键位？
- [ ] **AVWAP**：在成本线附近？
- [ ] **价格行为**：有 SFP/假突破？
- [ ] **多所对比**：Coinbase 现货确认？
- [ ] **OI 状态**：参与度如何？

**至少 3 个以上因子共振再入场！**

## ⚡ 快速判断规则

### 涨是真涨还是假涨？
```
现货买盘 + 合约买盘 = ✅ 真涨
仅合约买盘 = ⚠️ 可能骗炮
```

### 该不该入场？
```
蓝色信号 + 入场模型 + CVD 确认 = ✅ 入场
只有单一信号 = ❌ 等待
信号不够明确 = ❌ 宁可错过
```

### 该不该止盈？
```
CVD 归位 80%+ = ✅ 大部分止盈
CVD 完全重置 = ✅ 全部离场
目标价到了但 CVD 未归位 = 🤔 部分止盈
```

### 被套空单在下面 = 支撑？
```
只要现货不继续砸 → 空头需要买入平仓 → 支撑有效
现货持续砸盘 → 空头被解套 → 支撑消失
```
