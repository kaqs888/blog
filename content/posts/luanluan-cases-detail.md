# 案例一：蓝色信号抄底 —— 2025年11月8日

## 背景

BTC 从高位回调，价格跌至前低附近（约 $100,000 区域），市场恐慌情绪升温。

## 分析过程

### Step 1：订单簿观察
> "前低下方的现货和合约挂单墙（蓝色信号的原因）#BTC 蓝色信号中线胜率 85%+"

**使用指标：订单簿深度分析、蓝色信号**

- 价格接近前低时，订单簿显示买方挂单极其厚实
- 现货和合约的买方挂单都很大 → 蓝色信号触发
- 蓝色信号代表：买方挂单差值达到阈值

### Step 2：CVD 确认
> "这么多多头冲进去价格就上去一点点，还是觉得多头被吸收... 纯靠合约拉没有现货支持就是差点意思。"

**使用指标：CVD（累积成交量差值）**

- 虽然蓝色信号出现，但空头并未完全力竭
- 反弹主要靠合约买盘，现货参与不足
- → 需要等待更好的确认

### Step 3：综合判断
> "空也不太符合入场模型，再蹲一蹲看有没有更好的信号。"

**使用指标：入场模型**

- 蓝色信号 ✅
- 现货买盘配合 ❌
- 入场模型触发 ❌
- **结论：暂不入场，继续观察**

## 关键教训

**蓝色信号是重要参考，但不是"闭眼买"的信号。**
- 需要配合现货买盘确认
- 需要入场模型共同触发
- 宁可错过反弹也不盲目入场

## 最终结果

蓝色信号出现后，BTC 确实从前低反弹，但反弹力度不大，验证了"缺少现货支持"的判断。

# 案例二：CVD 阈值判断反转 —— 2025年11月7日

## 背景

BTC 处于区间震荡，价格持续下跌，CVD 不断走低。

## 分析过程

### Step 1：CVD 追踪
> "现在 #BTC 下来这波累积 delta -1.5b... 重新提示一下 10.11 之后，区间震荡反转的阈值大概在 -2b，破区间之后力竭的阈值大概 -3b~-4b（极端）。"

**使用指标：CVD 阈值系统**

- 当前 CVD = -1.5b，还未到 -2b 反转阈值
- 需要继续观察是走向 -2b（区间反转）还是 -3b（破区间）

### Step 2：多空对峙观察
> "多头入场 1000 个 BTC，空头入场 1500 个 BTC，多空都顶子弹上膛了。"

**使用指标：分交易所持仓量**

- 空头 (1500 BTC) > 多头 (1000 BTC) → 空头略占优
- 但双方都在加仓 → 大行情即将来临

### Step 3：SFP 判断
> "又回到了这个看这波是走 -2b 还是 -3b 的时候... 如果 -2b 反转，则 SFP 确认底部。"

**使用指标：SFP（假突破形态）**

- 如果价格在 -2b 附近形成假突破 → 反转信号
- 如果 CVD 继续走到 -3b → 趋势继续向下

## 交易模型

```
判断逻辑：
CVD 当前 = -1.5b
→ 如果跌到 -2b 且出现 SFP → 做多
→ 如果跌到 -3b~-4b → 等待力竭后做多
→ 如果 CVD 不继续跌且价格反弹 → 观望（信号不明确）
```

## 关键教训

**CVD 阈值不是精确数字，是参考区间。** -2b 是大致的反转区域，可能在 -1.8b 就反转，也可能到 -2.5b。配合 SFP 等确认信号使用更有效。

# 案例三：Coinbase 大额挂单支撑 —— 2026年1月17日

## 背景

BTC 价格在 $95,000 附近来回震荡，Coinbase 出现罕见的大额现货挂单。

## 分析过程

### Step 1：挂单异常发现
> "下不去了… Coinbase 下面挂了 400 个！一根线 50 个，最下面一根 80 个。"

**使用指标：订单簿深度（分交易所）**

- Coinbase 现货买方挂出 400 BTC 的大额买单
- 每个价位 50 BTC，最底部 80 BTC
- 这种规模在 Coinbase 极为罕见

### Step 2：Spoofing 排查
> "Coinbase 的流动性，这种大规模挂 400 个很少见且反常... 观察到这种挂单随着价格的波动，不停的在各个价位来回，疑似 spoofing。"

**使用指标：订单簿行为分析**

- 挂单在不同价位间移动 → 疑似 Spoofing
- 但 Coinbase 现货 Spoofing 较少见（主要在合约市场）
- 需要继续观察是否真的被成交

### Step 3：被套空单分析
> "95k 的挂单合约也没打掉，现货也还没打掉，但是昨天的低点打了 SFP 了（假突破）。"

**使用指标：被套单分析、SFP**

- $95k 处有大量空头被套
- 前低形成 SFP → 支撑有效的信号
- 但 Bybit 散户在这里抄底做多 → 需要警惕

### Step 4：止损获取流动性
> "空头止损的买盘跟被动的挂单卖盘成交... 有点经典的打止损获取流动性的操作了。"

**使用指标：成交明细分析**

- 价格上冲时，触发了空头的止损单
- 这些止损单形成了"被动买盘"
- 大户利用这些流动性出货

## 完整分析思路

```
1. Coinbase 出现 400 BTC 大额买单 → 不寻常
2. 挂单移动行为 → 可能是 Spoofing
3. 前低 SFP 形成 → 技术面有支撑信号
4. 但 Bybit 散户做多 → 可能被反向收割
5. 空头止损被打 → 大户获取流动性

综合判断：短期震荡，大额挂单可能是护盘但也可能是诱导
→ 不追多，等待更明确的信号
```

# 案例四：CVD 重置止盈策略 —— 2026年3月3日

## 背景

BTC 从高位开始下跌，水博乱乱在高位建立了空头仓位。

## 分析过程

### Step 1：做空入场
已有空头仓位，现在需要判断何时止盈。

### Step 2：现货出货确认
> "现货开始出货。"

**使用指标：现货 CVD、现货流动方向**

- 现货市场出现大量卖出
- 验证空头逻辑正确

### Step 3：CVD 重置作为止盈信号
> "现在这波空准备分阶段 TP，应该会在现货 CVD 重置的时候全部走掉... 无论那会价格是多少。"

**使用指标：CVD 重置**

- 从做空开始时的 CVD 水平算起
- 等待 CVD 回到（重置到）开始水平
- CVD 重置 = 卖方力量释放完毕 = 止盈时机

### Step 4：分阶段执行
> "昨天说了这个空会在 CVD 重置归位的时候全部走掉... 现在目前已经归位了 80% 了... 也相应的，66.6k 把部分仓位止盈了。"

**止盈进度：**

| CVD 归位 | 止盈比例 | 价格 |
|---------|---------|------|
| 80% | 第一批 | $66,600 |
| 100% | 全部 | 等待中 |

## 关键教训

**CVD 重置是一个优秀的止盈策略：**
1. 不依赖主观判断"够不够跌"
2. 用客观数据判断"卖方力量释放程度"
3. 分阶段止盈，避免贪心
4. "无论那会价格是多少" → 纪律性极强

# 案例五：CME 开盘入场模型 —— 2026年3月9日

## 背景

周末 BTC 价格下跌，周一 CME 即将开盘。

## 分析过程

### Step 1：CME 开盘效应
> "蹲了一个周末给我蹲到了 CME 开盘的这一下出了个入场模型... 这种疑似都是周末下跌之后 CME 低开，一些被迫平仓止损的激进多头。"

**使用指标：CME 缺口效应**

- 周末 BTC 下跌
- CME 周一低开 → 持仓多头被迫止损
- 止损带来的卖压 = 最后一波强制卖出

### Step 2：订单簿支撑确认
> "现在币安 65.6k~66k 被套了 1000+ 的空单... 只要现货不出货，这波空单就是支撑。"

**使用指标：被套单分析、订单簿**

- $65.6k-$66k 有 1000+ BTC 的空头被套
- 这些空头 = 潜在买方（需要买入平仓）
- 现货不出货 → 支撑仍然有效

### Step 3：入场模型触发
```
✅ CME 低开导致被迫止损 → 恐慌性卖出接近尾声
✅ 被套空单 1000+ BTC → 天然支撑
✅ 入场模型触发
→ 做多入场
```

## 完整思路链

```
周末 BTC 下跌
→ 预判 CME 低开会出现止损盘
→ 蹲守等待 CME 开盘
→ CME 低开 + 止损盘出现
→ 入场模型触发
→ 同时确认下方有 1000+ 空头被套做支撑
→ 做多入场
```

# 案例六：多指标共振盘面分析 —— 2026年2月10日

## 背景

BTC 刚经历暴跌至 $60,000，反弹至 $72,000，现在处于震荡整理阶段。

## 分析过程

### Step 1：Volume Profile 确定区间
> "今日盘面... 图 1 目前在插完 6W 之后到现在的这个 VAL 和 VAH 之间运作... 还在等待方向。"

**使用指标：Volume Profile（VAL/VAH）**

- 价格在 VAL 和 VAH 之间运行
- 还未选择方向

### Step 2：AVWAP 收敛确认
> "同时受到上下两根 AVWAP 的收敛... 分别是 1 月 31 号插 7W5 的那根和 6W 针尖的那根... 这个收敛最后一定会出方向。"

**使用指标：AVWAP**

- 从 $75,000 高点画的 AVWAP 向下
- 从 $60,000 低点画的 AVWAP 向上
- 两线收敛 → 大方向即将确定

### Step 3：OI 与现货驱动判断
> "从 OI 上看现在也从接近 60k 那会的历史高位回到了现在近期的相对低位... 多空都已撤离市场，这两天的震荡基本都是现货在驱动。"

**使用指标：OI（持仓量）**

- OI 从高位降至低位 → 杠杆去化
- 现货驱动 → 波动相对健康
- 等待新资金入场带来方向

### Step 4：综合判断
```
VAL-VAH 区间运行 → 等待方向
两条 AVWAP 收敛 → 大行情前兆
OI 低位 → 新方向启动时波动会很大
现货驱动 → 当前波动可信

操作建议：等待 AVWAP 收敛突破方向确定后跟随
```
