## 1. 订单簿不平衡色带

### 色带位置

| 位置 | 含义 |
|------|------|
| K 线下方 | **买单不平衡** — 该价位买盘 > 卖盘 |
| K 线上方 | **卖单不平衡** — 该价位卖盘 > 买盘 |

### 色带颜色

| 颜色 | 阈值等级 | 信号强度 | 含义 |
|------|----------|----------|------|
| 🟢 **绿色** | Level 1 | ⭐ | 轻度不平衡，参考性弱 |
| 🟡 **黄色** | Level 2 | ⭐⭐ | 中等不平衡，值得关注 |
| 🔴 **红色** | Level 3 | ⭐⭐⭐ | 强烈不平衡，重要支撑/压力 |
| 🔵 **蓝色** | Level 4 | ⭐⭐⭐⭐ | 极端不平衡，中线信号 |

### 实战解读

**蓝色色带在下方**：
> "前低下方的现货和合约挂单墙（蓝色信号的原因）"

→ 极强买单支撑，中线做多信号

## 2. CVD 曲线

### 线条含义

| 线条 | 颜色 | 代表 |
|------|------|------|
| 现货 CVD | 🔵 蓝色 | 三大所现货主动买卖差累积 |
| 合约 CVD | 🟠 橙色 | 合约主动买卖差累积 |

### 背离图示

**牛背离**（看多）：
```
价格：  ╲_____╱   （下降后横盘）
CVD：   ╲╲╲       （持续下降）
→ 抛压持续但价格不跌 = 被动买盘吸收
```

**熊背离**（看空）：
```
价格：  ╱‾‾‾‾╲   （上升后横盘）
CVD：   ╱╱╱       （持续上升）
→ 买盘增加但推不动价格 = 上方卖压重
```

### CVD 重置

```
极端 CVD → 逐步归位 → 趋势结束
-3b → -2.4b → -1.8b(平50%) → -0.6b(平80%) → 0(全平)
```

## 3. 订单簿深度图

### 图表结构

```
价格轴
  │  ████████  ← 卖单挂单（红/橙色柱）
  │──────────── ← 当前价格
  │  ██████████████ ← 买单挂单（绿色柱，更厚=更强支撑）
  └─── 挂单数量
```

### 颜色

| 颜色 | 含义 |
|------|------|
| 🟢 绿色 | 买单挂单（Bid） |
| 🔴 红色 | 卖单挂单（Ask） |
| 🔵 蓝色高亮 | 极大量挂单（异常） |

## 4. Delta / OI 图表

### Delta 柱状图

| 颜色 | 含义 |
|------|------|
| 🟢 绿色柱 | 正 Delta（主动买 > 主动卖） |
| 🔴 红色柱 | 负 Delta（主动卖 > 主动买） |

### 累积 Delta 关键阈值

| 阈值 | 含义 |
|------|------|
| ±1.5b | 趋势推进中 |
| ±2b | 区间反转阈值 |
| ±3b~4b | 极端力竭 |

## 5. 成本区间图

### Volume Profile

```
价格 │
     │  ████              ← 低成交量（易突破）
     │  ██████████████    ← POC（成交最密集）
     │  ████████          ← 高成交量
     │  ███               ← 低成交量
     └── 成交量
```

### AVWAP 线条

| 线条 | 含义 |
|------|------|
| 从高点锚定 | 压力线 |
| 从低点锚定 | 支撑线 |
| 多条交叉点 | 成本平衡点（强支撑/压力） |

### VAL / VAH

```
──── VAH ──── ← 突破 = 看多
│  震荡区间   │
──── VAL ──── ← 跌破 = 看空
```

## 图表快速检查清单

看到图表时快速检查：

1. ✅ 色带在上方还是下方？→ 支撑 or 压力
2. ✅ 色带什么颜色？→ 绿 < 黄 < 红 < 蓝
3. ✅ CVD 和价格是否背离？→ 背离 = 反转信号
4. ✅ Delta 到力竭阈值了吗？→ ±2b~4b 注意反转
5. ✅ 哪个交易所有异常挂单？→ Coinbase 大单 = 机构
6. ✅ 被套仓位在哪？→ 空单=支撑，多单=压力
7. ✅ AVWAP/POC 交汇处？→ 成本平衡点 = 强支撑/压力
