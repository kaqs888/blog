## TRDR 设置

TRDR 是最核心的工具，将订单簿数据可视化到 K 线图上。

**核心功能**：
- 订单簿不平衡色带（绿/黄/红/蓝）
- 5% 和 2.5% 订单簿差值
- 多交易所对比
- 大单追踪

**建议**：保持默认阈值，蓝色信号出现时重点关注。

## TradingView 设置

### 必装指标

**AVWAP（Anchored VWAP）**：
1. 选择锚定点（重要高低点）
2. 右键 → Add Anchored VWAP
3. 同时添加多条（不同锚定点）

**Volume Profile**：
1. 使用 Fixed Range Volume Profile
2. 关注 POC 线（黄色）
3. 设置 VAH/VAL（白色虚线）

**CVD**：推荐 LuxAlgo 的 Cumulative Volume Delta

### 推荐布局

```
┌─────────────────┐
│ K 线 + AVWAP    │
│ + Volume Profile│
├─────────────────┤
│ CVD 指标        │
├─────────────────┤
│ Delta / OI      │
└─────────────────┘
```
