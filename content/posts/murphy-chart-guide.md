## 一、MVRV 相关图表

### 1.1 MVRV 背离分析图

**典型图表**：

![MVRV 背离分析](/images/murphy/mvrv-backdivergence.jpg)

**线条说明**：

| 线条/区域 | 颜色 | 含义 |
|----------|------|------|
| **价格线** | 白色/主图 | BTC 现货价格 |
| **MVRV 线** | 副图曲线 | MVRV 比率值 |
| **MVRV 前高** | 虚线 | 上一轮周期 MVRV 高点 |
| **背离标注** | 箭头 | 价格新高但 MVRV 未新高 |

**识别方法**：
- 价格创新高 → 看主图
- MVRV 未创新高 → 看副图
- 两者背离 → 趋势衰减信号

---

### 1.2 MVRV 公允价值图

**典型图表**：

![MVRV 公允价值](/images/murphy/mvrv-fair-value.jpg)

**线条说明**：

| 线条/区域 | 颜色 | 含义 |
|----------|------|------|
| **BTC 价格** | 主图曲线 | 现货价格 |
| **公允价值线** | 水平线 | MVRV 历史平均值对应价格 |
| **低估区** | 绿色区域 | 价格 < 公允价值 |
| **高估区** | 红色区域 | 价格 > 公允价值 |

**解读**：
- 价格在公允价值下方 → 低估，可买入
- 价格在公允价值上方 → 高估，注意风险

---

### 1.3 MVRV 极端偏差定价区间

**典型图表**：

![MVRV 极端偏差定价区间](/images/murphy/mvrv-deviation-bands.jpg)

**线条说明**：

| 线条 | 颜色 | 含义 | 操作 |
|------|------|------|------|
| **紫线** | 紫色 | 极度高估区 | 突破后大概率回调 |
| **红线** | 红色 | 高估区 | 注意风险 |
| **橙线** | 橙色 | 中间位置 | 回调支撑/反弹阻力 |
| **黄线** | 黄色 | 偏低估区 | 可考虑分批买入 |
| **绿线** | 绿色 | 低估区 | 强支撑，投资价值高 |
| **价格线** | 白色 | BTC 价格 | 当前位置 |

**实战用法**：
- 价格触及紫线/红线 → 止盈
- 价格触及绿线/黄线 → 买入
- 橙线作为中间参考

## 二、筹码结构图表

### 2.1 URPD 巨量柱图

**典型图表**：

![筹码结构巨量柱](/images/murphy/urpd-huge-volume.jpg)

**图表类型**：热力图/柱状图

**元素说明**：

| 元素 | 颜色/形状 | 含义 |
|------|----------|------|
| **横轴** | - | 价格（从低到高） |
| **纵轴** | - | 时间（从下到上） |
| **巨量柱** | 亮色/高柱 | 大量筹码堆积区 |
| **普通区域** | 暗色 | 正常成交量 |
| **当前价格线** | 水平线 | 当前 BTC 价格 |

**解读**：
- 巨量柱位置 → 强支撑/阻力
- 巨量柱不是顶 → 大量资金承接
- 缺口区域 → 最终会被回补

---

### 2.2 URPD 缺口图

**典型图表**：

![URPD 缺口](/images/murphy/urpd-gap.jpg)

**元素说明**：

| 元素 | 含义 |
|------|------|
| **筹码密集区** | 亮色区域 |
| **缺口** | 暗色空白带 |
| **缺口上沿** | 阻力位 |
| **缺口下沿** | 支撑位 |
| **回补箭头** | 价格会回到缺口区域 |

**Murphy 理论**：
> "URPD 上所有的缺口都会被补齐"

---

### 2.3 双锚结构图

**典型图表**：

![双锚结构](/images/murphy/dual-anchor.jpg)

**线条说明**：

| 线条 | 含义 |
|------|------|
| **上锚点** | 高位筹码堆积区（阻力） |
| **下锚点** | 低位筹码堆积区（支撑） |
| **价格运行轨迹** | 在两个锚点之间波动 |
| **当前价格** | 靠近哪个锚点 |

**实战用法**：
- 价格接近下锚 → 买入
- 价格接近上锚 → 止盈
- 突破任一锚点 → 趋势延续

### 3.2 SPI 情绪压力指数

**典型图表**：

![SPI 倒挂](/images/murphy/spi-inversion.jpg)

**线条说明**：

| 线条 | 颜色 | 含义 |
|------|------|------|
| **多头情绪线** | 绿色 | 多头情绪强度 |
| **空头情绪线** | 红色 | 空头情绪强度 |
| **价格线** | 白色 | BTC 价格（副图） |

**信号解读**：

| 形态 | 含义 |
|------|------|
| **绿线上升 + 红线贴底** | 趋势启动 |

| **倒挂（红超绿）** | 空头反超，风险预警 |

## 四、其他核心指标

### 4.1 AVWAP 多线汇聚图

**典型图表**：

![AVIV 背离](/images/murphy/aviv-divergence.jpg)

**线条说明**：

| 线条 | 锚定点 | 含义 |
|------|--------|------|
| **AVWAP 1** | 6W 针尖 | 从 6W 低点计算的平均成本 |
| **AVWAP 2** | 6.25W 针尖 | 从 6.25W 低点计算的平均成本 |
| **AVWAP 3** | 突破点 | 从区间突破位计算 |
| **汇聚区** | 多线重叠 | 成本平衡点，激烈博弈 |

**识别方法**：
- 从 TradingView 的 AVWAP 工具
- 选择"针尖"作为锚定点
- 多条线汇聚 = 关键位置

### 4.2 CVD 累积成交量差值图

| 线条 | 含义 |
|------|------|
| **CVD 线** | 主动买入 - 主动卖出的累积值 |
| **零轴** | CVD=0，买卖平衡 |
| **上升段** | 买方主导 |
| **下降段** | 卖方主导 |
| **归位** | CVD 回到零轴附近 |

**Murphy 用法**：
> "CVD 归位 80% 时平仓 80%，保留 20% 观察"

---

## 五、快速识别指南

### 5.1 颜色速查表

| 颜色 | 常见含义 |
|------|----------|
| **红色** | 高估/风险/空头 |
| **绿色** | 低估/机会/多头 |
| **蓝色** | 中性/参考线 |
| **橙色** | 中间位置/警示 |
| **紫色** | 极端位置 |

### 5.2 线条类型

| 线条类型 | 含义 |
|----------|------|
| **实线** | 实际数据 |
| **虚线** | 参考位/前高前低 |
| **箭头** | 标注重点 |
| **阴影区** | 特殊区域（高估/低估） |

## 六、图表阅读流程

### 6.1 5 步读图法

```
1. 看标题 → 这是什么指标？
   ↓
2. 看坐标 → 横轴时间，纵轴数值
   ↓
3. 看图例 → 每条线/颜色代表什么
   ↓
4. 看当前值 → 价格在什么位置
   ↓
5. 看信号 → 高估/低估/背离/突破
```

### 6.2 Murphy 图表检查清单

- [ ] 识别所有线条含义
- [ ] 确认当前价格位置
- [ ] 检查有无背离
- [ ] 标记关键支撑/阻力
- [ ] 对比多个指标验证

## 七、TradingView 设置指南

### 7.1 添加 Murphy 常用指标

**步骤**：

1. 打开 TradingView
2. 点击"指标"按钮
3. 搜索并添加：
   - `MVRV Ratio`
   - `URPD`（需自定义脚本）
   - `AVWAP`（内置）
   - `CVD`（需自定义脚本）
   - `Investor Confidence`（需自定义）

### 7.2 AVWAP 锚定点设置

```
1. 选择 AVWAP 工具
2. 点击"针尖"位置（高低点）
3. 自动生成平均成本线
4. 多条 AVWAP 汇聚 = 关键位
```
