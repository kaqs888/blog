# 一、MVRV 极端偏差定价区间图

## 1.1 图表结构

```
┌──────────────────────────────────────────────────┐
│                                                  │
│  BTC 价格（纵轴，对数刻度）                       │
│                                                  │
│  $150,000 ── ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ 红线（+2σ）   │
│                          ↗                       │
│  $130,000 ── ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ 橙线（+1σ）   │
│                     ╱╲                           │
│  $110,000 ──      ╱  ╲                           │
│                  ╱    ╲  价格走势（白色/黑色）     │
│  $90,000  ── ─ ─╱─ ─ ─╲─ ─ ─ ─ ─ 黄线（均值）   │
│               ╱        ╲                         │
│  $70,000  ── ─ ─ ─ ─ ─ ─╲─ ─ ─ ─ 绿线（-1σ）   │
│                           ╲                      │
│  $50,000  ── ─ ─ ─ ─ ─ ─ ─╲─ ─ ─ 蓝线（-2σ）   │
│                                                  │
│  ──────────────────────────────────────────      │
│  时间（横轴）                                     │
└──────────────────────────────────────────────────┘
```

## 1.2 各线含义

| 线条 | 颜色 | 统计含义 | 交易含义 |
|------|------|---------|---------|
| +2σ | 🔴 红色 | 均值+2倍标准差 | 极度高估，周期顶部区域 |
| +1σ | 🟠 橙色 | 均值+1倍标准差 | 上方阻力位 |
| 均值 | 🟡 黄色 | 历史均值 | 中性位置，支撑/阻力切换 |
| -1σ | 🟢 绿色 | 均值-1倍标准差 | 下方支撑位 |
| -2σ | 🔵 蓝色 | 均值-2倍标准差 | 极度低估，周期底部区域 |
| 价格 | ⚪ 白色 | BTC 实际价格 | 与各线的相对位置决定判断 |

## 1.3 如何读取信号

| 价格位置 | 信号 | Murphy 的操作建议 |
|---------|------|------------------|
| 触碰红线 | 🔴 极端过热 | 考虑大幅减仓 |
| 接近橙线 | 🟠 上方阻力 | 注意回调风险 |
| 橙线-黄线之间 | 正常牛市 | 持有 |
| 接近黄线 | 中性支撑 | 观察是否守住 |
| 黄线-绿线之间 | 弱势 | 轻仓、谨慎 |
| 接近绿线 | 🟢 超跌 | 考虑加仓 |
| 触碰蓝线 | 🔵 极端低估 | 重仓买入区 |

# 二、URPD 筹码分布图

## 2.1 图表结构

```
┌──────────────────────────────────────────────────┐
│                           URPD                    │
│  价格（纵轴）                                     │
│                                                  │
│  $120,000 ── ████                                │
│  $115,000 ── ██████████████  ← 上方筹码堆积区(A) │
│  $110,000 ── ████████████████████████████████    │
│  $105,000 ── ████████                            │
│  $100,000 ── ███  ← 筹码稀疏区（"缺口"）         │
│  $ 95,000 ── ██████████████████████████          │
│  $ 90,000 ── ████████████████████               │
│  $ 85,000 ── ██████                              │
│  $ 80,000 ── ██████████████████                  │
│  $ 75,000 ── ████                                │
│  $ 68,000 ── ██████████████████████████████ ← B  │
│  $ 65,000 ── ████████████████████████████████████│
│              ← 巨量柱！下方筹码堆积区             │
│                                                  │
│  ──────────────────────────────────────────      │
│  BTC 数量（横轴）                                 │
│                                                  │
│  A 区和 B 区 = "双锚结构"                         │
│  中间的稀疏区 = 价格容易快速穿越                   │
└──────────────────────────────────────────────────┘
```

## 2.2 各元素含义

| 元素 | 含义 | 交易意义 |
|------|------|---------|
| 长条（宽柱）| 该价格上有大量 BTC | 强支撑或阻力 |
| 短条（窄柱）| 该价格上 BTC 很少 | 价格容易穿越 |
| 巨量柱 | 某个价格特别突出 | 潜在底部或顶部 |
| 红色（亏损）| 这些 BTC 成本高于当前价 | 持有者在亏损 |
| 绿色（盈利）| 这些 BTC 成本低于当前价 | 持有者在盈利 |

## 2.3 双锚结构的识别

**什么是有效的双锚结构：**
1. 上下各有一个明显的筹码堆积区
2. 中间有相对稀疏的区域
3. 历史验证有效性高

**Murphy 的观察：**
> "URPD 的双锚结构的支撑，在多轮周期多次行情回调中都得以检验其有效性。"

## 2.4 筹码变化的动态观察

Murphy 经常对比不同日期的 URPD：

```
日期对比：
11/19                     11/21
$97,000 ██████████████   $97,000 ██████████
$95,000 ████████████     $95,000 ████████████████
$90,000 ██████           $90,000 ██████████████ ← 筹码增加！
$85,000 ████             $85,000 ████████ ← 新买入

→ 说明 $90,000-$95,000 有资金在买入
```

# 三、BRS 信号图

## 3.1 图表结构

```
┌──────────────────────────────────────────────────┐
│  主图：BTC 价格                                   │
│       ╱╲         ╱╲                              │
│      ╱  ╲       ╱  ╲                             │
│     ╱    ╲     ╱    ╲                            │
│    ╱      ╲   ╱      ╲                           │
│   ╱        ╲ ╱        ╲╱                         │
├──────────────────────────────────────────────────┤
│  副图：BRS 指标（0-100）                          │
│                                                  │
│  100 ── ●     ●           ← BRS=100 买入信号！    │
│   80 ──                                          │
│   60 ──   │     │                                │
│   40 ──   │     │                                │
│   20 ──   │     │                                │
│    0 ── ──●─────●── ← BRS=0 卖出信号             │
│                                                  │
│  买入 → 持有 → 卖出 = 一次完整波段                │
└──────────────────────────────────────────────────┘
```

## 3.2 信号读取方法

| BRS 值 | 含义 | 操作 |
|--------|------|------|
| **100** | **极度超跌** | **考虑买入** |
| 50-99 | 下跌风险中等 | 等待 |
| 1-49 | 风险较低 | 观望 |
| **0** | **反弹结束/过热** | **考虑卖出** |

## 3.3 实际案例示意

```
时间线：

2026.02.02  BRS = 100 ✅ 买入信号
            BTC ≈ $75,000-$78,000
                    │
2026.02.07  BRS 仍然 = 100（定格不动）
            BTC 跌至 $60,000 后反弹至 $70,000
            "BRS仍然被定死在定格位置，纹丝不动"
                    │
2026.02.??  BRS 开始回落
                    │
2026.02.??  BRS = 0 → 卖出信号
```

**注意：** 这次交易中 BRS=100 买入后价格继续下跌，说明信号不是 100% 准确。

│      ╱    ╲  ╱╲                                  │
│  ─ ╱──────╲╱──╲────── 零轴                       │
│            ╲  ╱                                  │
│             ╲╱   ← 倒挂区域（空头>多头）          │
│  🔴 红线（空头情绪）                              │

## 4.2 各状态解读

| 状态 | 描述 | 市场阶段 |
|------|------|---------|
| 绿线高、红线低 | 多头主导 | 牛市趋势 |

| 绿线≈红线 | 多空均衡 | 震荡/过渡 |
| 红线>绿线 | 空头主导 | 超卖/熊市初期 |
| 红线远>绿线 | 极端倒挂 | 深熊/恐慌 |

# 综合图表阅读法

```

```
