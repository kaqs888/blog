## 第一章 基础概念篇（新手必读）

### 1.1 什么是链上数据（On-chain Data）？

**定义**：区块链上记录的所有交易数据，包括转账、持仓、地址活动等。

**与交易数据的区别**：
- **交易数据**：交易所的买卖记录（订单簿、成交量）
- **链上数据**：区块链上的真实转账（更真实，难以造假）

**为什么重要**：
- 反映真实资金流向
- 显示大户（鲸鱼）行为
- 揭示市场长期趋势

---

### 1.2 什么是 Glassnode？

**定义**：专业的链上数据分析平台，提供 BTC、ETH 等加密货币的链上指标。

**主要功能**：
- 链上指标计算（MVRV、NUPL、SOPR 等）
- 地址行为分析（鲸鱼追踪、持仓分布）
- 市场情绪指标（投资者信心、盈亏状态）

### 1.3 什么是 MVRV？

**定义**：Market Value to Realized Value Ratio，市场价值与已实现价值比率。

**计算公式**：
```
MVRV = 市值（Market Cap） / 已实现市值（Realized Cap）

市值 = 当前价格 × 流通供应量
已实现市值 = Σ(每个 BTC 最后一次移动时的价格)
```

**通俗理解**：
- 市值：当前市场总价值
- 已实现市值：所有人持仓成本的总和
- MVRV：当前价格相对平均成本的倍数

**如何解读**：
- **MVRV > 1**：市场整体浮盈（价格高于平均成本）
- **MVRV < 1**：市场整体浮亏（价格低于平均成本）
- **MVRV > 2**：高估区，注意风险
- **MVRV < 0.8**：低估区，投资价值高

---

### 1.4 什么是 URPD？

**定义**：UTXO Realized Price Distribution，UTXO 已实现价格分布。

**基本概念**：
- **UTXO**：Unspent Transaction Output，未花费交易输出
- 简单理解：每个 BTC 最后一次移动时的价格记录

**如何解读**：
- 显示每个价格档位有多少 BTC 持仓
- 巨量柱：某个价格有大量 BTC 换手
- 缺口：某个价格区间成交量很少

---

### 1.5 什么是 SOPR？

**定义**：Spent Output Profit Ratio，花费输出盈亏比率。

**计算公式**：
```
SOPR = 花费的 UTXO 总价值 / 这些 UTXO 的成本

SOPR > 1：整体盈利（人们在赚钱卖出）
SOPR < 1：整体亏损（人们在亏钱卖出）
```

**如何解读**：
- SOPR > 1 且上升：盈利增加，可能见顶
- SOPR < 1 且下降：亏损增加，可能见底
- SOPR = 1：盈亏平衡点

---

### 1.6 什么是 NUPL？

**定义**：Net Unrealized Profit/Loss，净未实现盈亏。

**计算公式**：
```
NUPL = (市值 - 已实现市值) / 市值

NUPL > 0：市场整体浮盈
NUPL < 0：市场整体浮亏
```

**颜色区域**：
- 🔴 **红色**（NUPL < 0）：恐慌/投降区（买入机会）
- 🟡 **黄色**（0 < NUPL < 0.25）：犹豫/乐观区
- 🟢 **绿色**（NUPL > 0.25）：贪婪/ euphoria 区（注意风险）

---

### 1.7 什么是已实现价格（Realized Price）？

**定义**：整个市场所有 BTC 的平均成本价。

**计算公式**：
```
已实现价格 = 已实现市值 / 流通供应量
          = Σ(每个 BTC 最后一次移动时的价格) / 总 BTC 数
```

**如何解读**：
- 价格 > 已实现价格：市场整体盈利
- 价格 < 已实现价格：市场整体亏损
- 是重要的支撑/阻力参考

---

### 1.8 什么是长期持有者（LTH）vs 短期持有者（STH）？

**定义**：
- **长期持有者（LTH）**：持仓超过 155 天的地址
- **短期持有者（STH）**：持仓少于 155 天的地址

**为什么重要**：
- LTH：通常是机构、大户，更聪明
- STH：通常是散户，更容易恐慌/贪婪

**关键指标**：
- LTH-SOPR：长期持有者的盈亏
- STH-SOPR：短期持有者的盈亏
- LTH 供应占比：长期持有者持有的 BTC 比例

---

### 1.9 什么是 MVRV 极端偏差定价区间？

**定义**：基于 MVRV 历史数据的概率统计通道。

**颜色分级**（从高到低）：
- 🟣 **紫线**：极度高估区（MVRV 极高）
  - 突破后大概率回调
- 🔴 **红线**：高估区
  - 注意风险
- 🟠 **橙线**：中间位置
  - 常为回调支撑或反弹阻力
- 🟡 **黄线**：偏低估区
  - 可考虑分批买入
- 🟢 **绿线**：低估区
  - 强支撑，投资价值高

## 第二章 估值类指标（MVRV 等）

### 2.1 MVRV（Market Value to Realized Value）

**定义**：市场价值与已实现价值的比率。

**Murphy 的核心洞察**：

> "MVRV 本质上代表资金和成本的相互关系。BTC 十多年来始终遵循 MVRV 和现货价格大级别背离的原则，即一旦出现价格更高但 MVRV 却更低的背离现象，后面指标就无法突破前高，随后价格的空间就被压制。"

**图表解读**：

**图表解读**：

  - 白色线：BTC 价格
  - 横轴：时间，纵轴（左）：价格（美元）
- 副图（下）：MVRV 指标
  - 橙色线：MVRV 值
  - 橙色虚线：MVRV 前高
  - 横轴：时间，纵轴（右）：MVRV 比率
- 背离信号：
  - 价格创新高（主图）
  - MVRV 未创新高（副图）
  - → 趋势动能衰减，可能回调

**使用方法**：
- **MVRV > 1**：市场整体浮盈，越高泡沫越大
- **MVRV < 1**：市场整体浮亏，投资价值区间
- **大级别背离**：价格新高但 MVRV 未突破前高 → 趋势动能衰减

**定义**：MVRV 历史累积平均值对应的价格。

## 第三章 筹码结构类指标（URPD 等）

### 3.1 URPD（UTXO Realized Price Distribution）

**Murphy 最核心的分析工具之一**，权重极高。

> "筹码结构在我们的分析体系中具有极高的权重，它对形成价格支撑/阻力有着重要的影响。"

**核心概念 —— 巨量柱**：

> "每当 BTC 的价格即将形成阶段性大底之前，都会在 URPD 的某个单一价格附近形成'巨量柱'。有巨量柱的地方，大概率不是周期的顶。因为在巨量柱的周围通常都会堆积大量的筹码，形成了一个价格范围的堆积区。"

  - 白色线：BTC 价格
  - 横轴：时间，纵轴（左）：价格（美元）
- 副图（右）：URPD 筹码分布
  - 热力图：颜色越亮 = 成交量越大
  - 巨量柱：某个价格的亮色垂直柱
  - 横轴：成交量，纵轴（右）：价格（美元）
- 信号解读：
  - 巨量柱位置 → 强支撑/阻力
  - 巨量柱不是顶 → 大量资金承接
  - 缺口区域 → 最终会被回补

## 第四章 实战案例

### 案例 1：MVRV 背离逃顶

**时间**：2025 年 5 月

**背景**：
- BTC 从 60k 上涨到 100k
- 市场情绪 euphoria

**信号出现**：
1. 价格创新高到 100k
2. MVRV 未突破前高（背离）
3. SOPR > 1.5（极度盈利）

**操作**：
- 在 100k 附近分批止盈
- 保留 20% 观察
- 等待 MVRV 回归公允价值

**结果**：
- BTC 回调到 80k
- 成功逃顶

**引用教程知识点**：
- 第二章：MVRV 背离
- 第二章：SOPR 极度盈利
- 第三章：筹码结构

**时间**：2025 年 11 月

**背景**：
- BTC 从 74k 下跌到 65k
- 市场恐慌

**信号出现**：
1. URPD 在 65k 形成巨量柱
2. MVRV < 1（低估区）
3. 投资者信心指数红色区域

**操作**：
- 在 65k 附近分批买入
- 止损：63k（巨量柱下方）
