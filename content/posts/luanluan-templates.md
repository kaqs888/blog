## 入场检查清单

入场前必须全部确认：

- [ ] 价格到达关键区间？（POC/AVWAP/订单簿色带）
- [ ] 订单簿显示不平衡？（色带颜色 ≥ 黄色）
- [ ] CVD 存在背离？
- [ ] Delta 接近力竭阈值？
- [ ] 入场模型质量如何？
- [ ] 美股状态？
- [ ] 有无宏观事件？
- [ ] 仓位空间充足？

### 模型质量评分

| 标准 | 是否满足 |
|------|----------|
| 价格到达关键位 | □ |
| 订单簿不平衡确认 | □ |
| CVD/Delta 信号确认 | □ |
| 多交易所数据一致 | □ |
| 无宏观风险事件 | □ |

**4-5 分** → 正常入场 | **3 分** → 头皮 | **1-2 分** → 放弃

## 止盈策略模板

### 头皮止盈

| 阶段 | 操作 |
|------|------|
| TP1 | 500-1000 点，平 50% |
| 触发 TP1 | 移保本 |
| TP2 | 前高/前低，平 30% |
| TP3 | 交给市场，平 20% |

### CVD 归位止盈

| CVD 归位进度 | 操作 |
|-------------|------|
| 50% | 减仓 30% |
| 80% | 减仓至 20% |
| 100% | 全部止盈 |

## 决策流程

```
判断大方向 → 检查订单簿色带 → 检查 CVD 背离 → 检查 Delta 力竭
                 │                    │                 │
           蓝色→中线入场        牛/熊背离→偏多/空    ±2b→反转可能
           红/黄→等确认          无背离→观察         ±3b~4b→强反转
           绿/无→观望                                 <±1.5b→延续
                                     │
                              模型质量评分
                           好→入场 | 差→放弃
```
